Display sub categories as a thumb in parent category page

== Description ==

When you click a category in woocommerce , then all products listed under that category . But if the category has some sub category then is not displayed . So Woocommerce Subcategory Thumb plugin will display the sub categories thumb as along with product . See the screenshot .

NB : wootheme compatible only .
